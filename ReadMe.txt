Read Me for probability app

All code is found in the main.ipynb file

7 projects are in the probability app
1. Single Die Roll-rolls a single die n times with the abillity to alter the number of rolls
2. Multi Dice Roll - rolls n dice for m trials and generates a fingure in real time
3. Coin Flip - Flips a single coin n times
4. Multi coin flip- Filps n coins for m trials 
5. 1D Random Walk - represents a waited random walk for n steps with x wait
6. 2D random walk - represents an unweighted 2 dimensional walk for n steps
7. Boltzman Equation - Calculates the boltzman equation
